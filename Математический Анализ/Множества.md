# Интервалы
```
[0, 1] = {x : 0 <= x <= 1
(0, 1) = {x : 0 < x < 1}
(0, 1] = {x : 0 < x <= 1}
[0, 1) = {x : 0 <= x < 1}
```

Интервалы используются в функциях

Функции. Последовательность. Инъекции, дисрекция, диекция. Обратно отображение. График функции, копмозиция функции, не явные задания функций, элементарные функции
